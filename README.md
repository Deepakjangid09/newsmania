# 📰 NewsMania

NewsMania is a modern and responsive web application that delivers the latest news from around the world in real-time. It fetches data from a news API and displays categorized news such as Technology, Sports, Business, Health, and Entertainment.

---

## 🚀 Features

* 🔥 Real-time news updates using API
* 📂 Category-based news filtering
* 🌐 Responsive design (Mobile + Desktop)
* ⚡ Fast loading with optimized API calls
* 🔍 Search functionality (optional if added)
* 🖼️ Clean and modern UI

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript / React.js
* API: News API (e.g., NewsAPI.org)
* Styling: CSS / Bootstrap / Tailwind (update as per your project)
* Deployment: Vercel / Netlify

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/newsmania.git
```

2. Navigate to project folder:

```
cd newsmania
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm start
```

---

## 🔑 API Configuration

* Get your API key from: https://newsapi.org/
* Create a `.env` file in root directory:

```
REACT_APP_NEWS_API_KEY=your_api_key_here
```
